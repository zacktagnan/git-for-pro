# Git for Pro

Aprendiendo a utilizar Git y GitHub de forma más profesional dentro de un proyecto Laravel.

## Requisitos y/o Funcionalidades Empleadas

- [Docker](https://www.docker.com/get-started/) (según el S.O.).
    - [WSL2](https://learn.microsoft.com/es-es/windows/wsl/install) para sistemas de Windows.
- [Laravel Sail](https://laravel.com/docs/12.x/sail).

## Instalación

````
git clone git@github.com:zacktagnan/git-for-pro.git
cd git-for-pro
sail up -d
````
